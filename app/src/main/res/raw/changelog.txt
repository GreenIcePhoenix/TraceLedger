# 1.0.0
[Repeat] Recurring Transactions :: Schedule automatic expenses, income, and transfers — daily, weekly, monthly, quarterly, or yearly.
[Lightbulb] Financial Insights :: Spending vs last month, savings rate, recurring cost summary, and net worth direction on the dashboard.
[PieChart] Spending Trends :: Category spend tracked across months with a line chart in Statistics.
[TrendingUp] Transaction History :: Transactions now grouped by date with Today / Yesterday headers and an amount range filter.
[Receipt] Transaction Detail :: Tap any transaction to view full details, edit, or delete directly from the detail sheet.
[Savings] Budget Warnings :: Budget alerts now trigger at 75% (warning) and 90% (critical) — not just when exceeded.
[Palette] Number Format :: Choose between Indian (1,00,000) and International (100,000) number formatting in Settings.
[Star] Onboarding :: First-launch walkthrough introducing the app for new users.
